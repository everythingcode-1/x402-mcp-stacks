# x402 Guard - Landing Page

Modern landing page untuk x402 Guard SDK.

## Tech Stack

- React 18 + TypeScript
- Vite (build tool)
- Tailwind CSS
- Framer Motion (animations)
- Lucide React (icons)

## Development

```bash
# Install dependencies
npm install

# Run dev server
npm run dev
```

Server akan berjalan di http://localhost:8080

## Build

```bash
npm run build
```

Output di folder `dist/`

## Features

- ✅ Hero section dengan code preview
- ✅ Features showcase
- ✅ SDK vs Marketplace comparison
- ✅ CTA ke Dashboard
- ✅ Responsive design
- ✅ Modern animations

## Deploy

Deploy ke Vercel, Netlify, atau Cloudflare Pages:

```bash
npm run build
# Upload dist/ folder
```
